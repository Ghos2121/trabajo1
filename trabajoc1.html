resumen c++

librerias

Las librerias son clases y funciones que se ponen al inicio del codigo, en generak son funcionas para usar, tambien es llamada como biblioteca


funciones y constantes

las funciones como su nombre lo dice son funciones que se le pueden perdir al programa y las constantes como su nombre lo dice son cosas que siempre estan ahi constantemente


gotoxy y main

el gotoxy es una de las principales funciones sirve para mover el curosr ya sea si esta en x o y, y el main es el inicio de cualquier programa


char y strin

char se usa para representar caracteres individuales ya sea: x y etc.. y strin es una cadena de caracteres


int y float

la funcion int se usa para introducir numeros de tipo entero y float es muy util ya que ayuda a poner un elemento fto etc al lado izquierdo o derecho


bool e if

la funcion se usa para representar datos de logica binaria y el if se usa para decir si o no depende de cuando se use


ciclos

los ciclos son algo esencial ya sea para c++ o diferentes lenguajes estos nos permiten repetir una o mas acciones cuantas veces lo necesitemos


vectores

los vectores nos permiten almacenar varios valores en una misma variable de hecho es un poco parecido a un ciclo
